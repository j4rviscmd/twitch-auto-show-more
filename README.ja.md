# twitch-auto-show-more

[ホームページ](https://j4rviscmd.github.io/twitch-auto-show-more/)

## 説明

Twitchのチャットで「もっと見る」ボタンを自動的にクリックするChrome拡張機能です。

## 機能

- チャットの「もっと見る」ボタンを自動的にクリック

## インストール（Chrome Web Store）

準備中

## インストール（手動）

1. リポジトリをクローンまたはダウンロード
2. Chromeの拡張機能管理ページ（`chrome://extensions/`）を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このフォルダを選択

## 使い方

- Twitchを開くと自動的に動作します
- 拡張機能のアイコンをクリックしてON/OFFを切り替えられます

## 対応ブラウザ

- Google Chrome
- Microsoft Edge
- その他Chromiumベースのブラウザ
