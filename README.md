# twitch-auto-show-more

[Homepage](https://j4rviscmd.github.io/twitch-auto-show-more/)

## Description

A Chrome extension that automatically clicks the "Show More" button in Twitch chat.

## Features

- Automatically clicks "Show More" button in chat

## Installation(Chrome Web Store)

WIP

## Installation(Manual)

1. Clone or download this repository
2. Open Chrome Extensions page (`chrome://extensions/`)
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder

## Usage

- Works automatically when you visit Twitch
- Click the extension icon to toggle ON/OFF

## Supported Browsers

- Google Chrome
- Microsoft Edge
- Other Chromium-based browsers
